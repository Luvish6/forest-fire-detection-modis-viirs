# Notebooks

This directory is intended for Jupyter Notebooks (`.ipynb`) used for experimentation, prototyping, and interactive analysis.

Currently, the project logic has been modularized into Python scripts in the `src/` directory for better reproducibility and structure.

To create a new notebook:
1.  Launch Jupyter Notebook or Lab.
2.  Import modules from `src` (ensure `src` is in your python path).
